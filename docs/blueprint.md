# **App Name**: SmartExpense AI

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication.
- Expense Management: Create, edit, and delete expense records, categorized for easy tracking.
- Monthly Budget Configuration: Set and adjust monthly budgets to monitor spending habits.
- Dashboard Visualization: Display key financial metrics and interactive charts summarizing spending.
- AI-Driven Savings Suggestions: Provide intelligent recommendations for saving and expense control based on spending patterns. This feature will use business rules as a reasoning tool.
- Data Export: Allow users to export their expense data to a CSV file for external analysis.
- Firestore Integration: Store user data, expenses, and budgets securely using Firestore.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trustworthiness and stability.
- Background color: Light indigo (#E8EAF6) to create a calm and clean backdrop.
- Accent color: Cyan (#00BCD4) to highlight key interactive elements and important information.
- Body and headline font: 'Inter', a sans-serif font providing a modern, clean and objective feel suitable for both headlines and body text.
- Simple, clean icons to represent expense categories and actions.
- Clean, responsive layout optimized for both desktop and mobile devices.
- Subtle transitions and animations to enhance user experience.
